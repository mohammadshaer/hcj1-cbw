Aufgabe 4

  1. .1
    Relative Maßeinheiten

    Relative Maßeinheiten passen sich dynamisch an das Layout und die Größe anderer Elemente an. Sie sind flexibel und werden häufig in responsive Webdesigns verwendet.

    1. em:
    1. Bedeutung: Eine em-Einheit entspricht der Schriftgröße des übergeordneten Elements. Wenn das übergeordnete Element eine Schriftgröße von 16px hat, entspricht 1em 16px.
    2. Beispiel: 
                              html {
                            font-size: 16px;
                        }
                        .child {
                            font-size: 1.5rem; /* 24px */
                        }
    1. rem:
    1. Bedeutung: Eine rem-Einheit entspricht der Schriftgröße des Root-Elements (normalerweise das -Element). Wenn das Root-Element eine Schriftgröße von 16px hat, entspricht 1rem 16px.
    2. Beispiel: 
                              html {
                            font-size: 16px;
                        }
                        .child {
                            font-size: 1.5rem; /* 24px */
                        }
    Relative Maßeinheiten

    Absolute Maßeinheiten haben feste Werte, die unabhängig von anderen Elementen sind. Sie sind nützlich, wenn eine genaue Kontrolle über die Größe erforderlich ist.

    1. px (Pixel):
    1. Bedeutung: Ein Pixel ist ein fester Wert und entspricht einem Bildschirmpixel.
    2. Beispiel: 
                              .box {
                            width: 100px;
                            height: 50px;
                        }
    1. in (Inch):
    1. Bedeutung: Ein Zoll entspricht 96 Pixeln auf einem Bildschirm (1in = 96px).
    2. Beispiel: 
                              .box {
                            width: 1in; /* 96px */
                            height: 0.5in; /* 48px */
                        }
