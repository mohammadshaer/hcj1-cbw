  3. .3 -Die if..else-Verzweigung wird verwendet, um eine oder mehrere Bedingungen zu prüfen. Jede Bedingung wird einzeln geprüft, und der Codeblock innerhalb der passenden Bedingung wird ausgeführt.
    -Der switch-Case wird verwendet, um den Wert einer Variable gegen mehrere mögliche Werte zu prüfen. Jeder mögliche Wert hat einen eigenen Fall (case), und der Codeblock innerhalb des passenden Falls wird ausgeführt. Am Ende jedes Falles steht normalerweise ein break, um zu verhindern, dass die nachfolgenden Fälle ebenfalls ausgeführt werden.

              const pool = {
            Groesse_m2: 500,
            kapazitaet: 100, 
            wassertemperatur: 28, 
            besucher: 0,
 
            come: function(anzahl) {
                if (this.besucher + anzahl > this.kapazitaet) {
                    console.log("Der Bademeister wird nervös! Keine weiteren Gäste können eintreten.");
                } else {
                    this.besucher += anzahl;
                    console.log(anzahl + " Gäste sind gekommen. Aktuelle Besucherzahl: " + this.besucher);
                }
            },
            leave: function(anzahl) {
                if (this.besucher - anzahl < 0) {
                    console.log("Es sind nicht genug Besucher im Pool, um " + anzahl + " Gäste zu entfernen.");
                } else {
                    this.besucher -= anzahl;
                    console.log(anzahl + " Gäste sind gegangen. Aktuelle Besucherzahl: " + this.besucher);
                }
            }
        };
        pool.come(30);
        pool.come(90);
        pool.leave(15);
        pool.leave(100);
