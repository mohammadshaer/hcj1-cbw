CSS(Cascading Style Sheet)

Es gibt drei Ansätze, die Sie befolgen können, um CSS-Code hinzuzufügen

  1. Inline CSS:

    Jedem Tag, das Sie auf der Seite hinzufügen, können Sie die style-Eigenschaft hinzufügen und den CSS-Code direkt als Wert übergeben.

                                
    Beispiel:
    
                            
    
                            <h1 style="color:blue; text-align:center;">Hello World!</h1>

  2. Internal CSS:

    Sie können das Styling-Tag an einer beliebigen Stelle auf der Seite hinzufügen und darin CSS-Eigenschaften zu jedem Tag auf der Seite hinzufügen.

                                
    Beispiel:
    
                            
    
                                    <style>
                                        h1 {
                                            color: blue;
                                            text-align: center;
                                        } 
                                        </style>
                                        <h1>Hello World!</h1>

  3. External CSS

    Sie können den CSS-Code in die CSS-Erweiterungsdatei einfügen und ihn dann auf der Seite mit dem Tag Link einbetten, der im head-Tag platziert werden muss.

    Beispiel: 

                                    style.css
                                
    
                                    h1 {
                                        color: blue;
                                        text-align: center;
                                    } 
                                
    
                                index.html
                                
    
                                        <!DOCTYPE html>
                                        <html>
                                            <head>
                                                <!-- Hier haben wir die Datei style.css -->
                                                <link rel="stylesheet" href="style.css">
                                            </head>
                                            <body>
                                                <h1>Hello World!
                                            </body>
                                        </html>
